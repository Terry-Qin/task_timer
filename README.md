# Windows自动锁屏应用

这是一个Windows桌面应用程序,允许用户设置一个指定的时间后自动锁定屏幕。应用程序具有以下特点:

## 功能

- 图形用户界面,方便设置锁屏时间
- 用户可以选择任意分钟数作为倒计时
- 倒计时最后10秒会在屏幕上显示倒计时数字
- 时间到达后自动锁定Windows屏幕

## 技术栈

- 使用Python编写
- GUI框架: PyQt5
- 系统操作: ctypes库

## 安装

1. 确保您的系统已安装Python 3.7+
2. 克隆此仓库
3. 安装依赖:
   ```
   pip install -r requirements.txt
   ```
